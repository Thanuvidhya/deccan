# SkillBridge AI - Conversational Skill Assessment & Learning Agent

🚀 **Hackathon Project** - AI-powered skill assessment and personalized learning platform

## 📋 Project Overview

SkillBridge AI bridges the gap between claimed skills and actual proficiency through conversational assessment and personalized learning recommendations. Our platform helps individuals validate their skills and companies make better hiring decisions.

## 🎯 Key Features

### 🧠 AI-Powered Assessment Engine
- **Resume Analysis**: AI parses and understands candidate experience
- **Dynamic Question Generation**: Creates personalized interview questions
- **Real-time Evaluation**: Assesses responses for technical accuracy
- **Multi-dimensional Scoring**: Evaluates conceptual knowledge and practical application

### 💬 Conversational Interview Interface
- **Chat-based Assessment**: Natural, stress-free conversation format
- **Adaptive Questioning**: Questions adapt based on candidate responses
- **Real-time Feedback**: Immediate scoring and insights during assessment
- **15-minute Assessment**: Quick yet comprehensive evaluation

### 🎯 Intelligent Gap Analysis
- **Skill Mapping**: Maps candidate skills against job requirements
- **Proficiency Scoring**: Quantifies skill levels across multiple dimensions
- **Gap Prioritization**: Identifies critical vs. nice-to-have skill gaps
- **Visual Analytics**: Interactive charts and progress tracking

### 📚 Personalized Learning Engine
- **Curated Resources**: 500+ learning resources across technologies
- **Time Estimation**: Realistic learning timelines based on skill complexity
- **Learning Path Generation**: Step-by-step roadmap from current to target proficiency
- **Progress Tracking**: Monitor learning journey and skill improvement

## 🛠️ Tech Stack

### Frontend
- **React 18** with TypeScript for type safety
- **Tailwind CSS** for responsive, modern UI
- **React Router** for seamless navigation
- **Lucide Icons** for professional interface

### Architecture
- **Component-based design** with reusable UI components
- **Responsive layout** for mobile and desktop
- **Modern React patterns** with hooks and functional components
- **TypeScript** for better development experience

## 🚀 Getting Started

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd skillbridge-ai
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to view the application

### Build for Production

```bash
npm run build
```

## 🎨 User Journey

### Step 1: Onboarding
- User uploads resume and inputs target job description
- AI analyzes documents and identifies key skill requirements
- System generates personalized assessment plan

### Step 2: Conversational Assessment
- 15-minute chat-based interview with AI interviewer
- Real-time evaluation and feedback
- Adaptive questioning based on responses

### Step 3: Results & Analysis
- Comprehensive skill assessment report
- Visual gap analysis with priority scoring
- Strengths and weaknesses breakdown

### Step 4: Personalized Learning
- Custom learning roadmap with curated resources
- Realistic time estimates for skill acquisition
- Progress tracking and milestone achievements
